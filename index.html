Bootstrap vs raw CSS

02/19/2022

Two article entries in one day??? Wow!!!! I'm on a roll! I've been talking to some friends about whether or not my website should implement Bootstrap, or keep it pure handwritten CSS. I don't know how it's going to play with React since I barely know anything about the framework, but I have a feeling that its better for my learning that I stick to CSS. The point of this website is to learn as much as I can about web development and making my own stylesheet only helps hone my craft. The reasons why I might want to abandon CSS entirely is rather enticing, however. It's a lot easier to skim whatever button or nav-bar strikes your fancy and construct it by giving a tag some class names. I won't do that though, because CSS is rather fun for me (in a masochistic sort of way I guess). Maybe I haven't done enough to out right hate it the same way how I have a strong distaste to python coding. Only time will tell, and you'll be able to read me (or don't, it doesn't matter because I'll still do it anyways) in a future blog entry.
